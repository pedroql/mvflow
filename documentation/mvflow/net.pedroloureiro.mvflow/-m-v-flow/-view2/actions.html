actions

abstract fun actions(): Flow<Action> (source)

Function to return a kotlinx.coroutines.flow.Flow of the actions that the user makes inside this view.

You can create this in many ways, one suggestion is using FlowBinding or doing it yourself like in this sample:

class MyActivity : AppCompatActivity(), MVFlow.View<State> {
    // could also be a Fragment, or a plain class you created.
    private val actionChannel = Channel<Action>()

    fun setupUi() {
        swipeRefreshLayout.setOnRefreshListener {
            actionChannel.offer(Action.Load)
        }
    }

    override fun actions() = actionChannel.consumeAsFlow()
}
